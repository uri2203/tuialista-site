# tuialista-site

Sitio estático público de **TuIAlista** (landing, portal, panel admin, panel de afiliados)
servido por GitHub Pages en https://tuialista.com.

El backend (núcleo de licencias, agentes, API del asistente) y los borradores legales
viven en un repositorio **privado** aparte y **no** se publican aquí.
